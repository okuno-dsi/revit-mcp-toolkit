- カテゴリ: FireProtection

## 概要

## 使い方

- パラメータ: なし

### リクエスト例
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "delete_fire_protection_instance",
  "params": {}
}
```
