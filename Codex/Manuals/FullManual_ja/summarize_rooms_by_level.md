# summarize_rooms_by_level

- カテゴリ: Room
- 目的: このコマンドは『summarize_rooms_by_level』を集計します。

## 概要
このコマンドは JSON-RPC を通じて実行され、目的に記載の処理を行います。使い方のセクションを参考にリクエストを作成してください。

## 使い方
- メソッド: summarize_rooms_by_level

- パラメータ: なし

### リクエスト例
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "summarize_rooms_by_level",
  "params": {}
}
```
