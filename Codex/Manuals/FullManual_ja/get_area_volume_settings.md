# get_area_volume_settings

- カテゴリ: Room
- 目的: このコマンドは『get_area_volume_settings』を取得します。

## 概要
このコマンドは JSON-RPC を通じて実行され、目的に記載の処理を行います。使い方のセクションを参考にリクエストを作成してください。

## 使い方
- メソッド: get_area_volume_settings

- パラメータ: なし

### リクエスト例
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "get_area_volume_settings",
  "params": {}
}
```
