# populate_door_window_legend_from_template

- Category: AnnotationOps
- Purpose: (Currently disabled) Populate a Door/Window legend from a template.

## Status
This command currently returns `ok:false` immediately due to performance/time-out problems in heavy models.

Use these commands instead:
- `copy_legend_components_between_views`
- `layout_legend_components_in_view`
- `set_legend_component_type`

## Usage
- Method: `populate_door_window_legend_from_template`
