# Docs (入口)

この `Docs/` は **トップの入口**として用意しています。実体のマニュアルは既存の
`Codex/Manuals/` に集約されているため、まずは下記から辿ってください。

- 入口: `Codex/Manuals/README.md`
- 更新履歴: `Codex/Manuals/UpdateLog.md`
- Big Change（フォルダ構成）: `Codex/Manuals/Big_Change.md`
- 作業開始: `Codex/START_HERE.md`

※ リポジトリ構成は既存の参照関係を保つため **`Codex/Manuals` を維持**しています。
